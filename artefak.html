Rencana Pembelajaran Siklus 1
Variabel, Tipe Data, dan Operator (JavaScript)

Artefak ini membahas materi Variabel, Tipe Data, dan Operator dalam JavaScript sebagai dasar pemahaman pemrograman bagi peserta didik kelas X Rekayasa Perangkat Lunak (RPL) di SMKN 1 Surabaya. Materi ini dirancang untuk membantu peserta didik memahami konsep dasar dalam penulisan program, mulai dari cara menyimpan data menggunakan variabel, mengenali berbagai jenis tipe data yang digunakan dalam JavaScript, hingga memanfaatkan operator untuk melakukan perhitungan, perbandingan, dan pengolahan data. Melalui pembelajaran ini, peserta didik diharapkan mampu membangun fondasi yang kuat dalam pemrograman sehingga dapat mengikuti materi yang lebih kompleks pada tahap pembelajaran selanjutnya.

Rencana Pembelajaran Siklus 2
Percabangan, Perulangan, dan Function (JavaScript)

Artefak ini membahas materi Percabangan, Perulangan, dan Function dalam JavaScript sebagai bagian dari pembelajaran dasar pemrograman bagi peserta didik kelas X Rekayasa Perangkat Lunak (RPL) di SMKN 1 Surabaya. Materi ini dirancang untuk membantu peserta didik memahami cara mengendalikan alur program menggunakan struktur percabangan, mengotomatisasi proses yang berulang melalui perulangan, serta mengorganisasi kode secara lebih efektif dengan menggunakan function. Melalui pembelajaran ini, peserta didik diharapkan mampu mengembangkan logika berpikir komputasional dan menerapkan konsep dasar pemrograman untuk menyelesaikan berbagai permasalahan secara sistematis dan efisien.

Rencana Pembelajaran Siklus 3
Array, Object, dan DOM (JavaScript)

Artefak ini membahas materi Array, Object, dan Document Object Model (DOM) dalam JavaScript sebagai bagian dari pembelajaran pemrograman bagi peserta didik kelas X Rekayasa Perangkat Lunak (RPL) di SMKN 1 Surabaya. Materi ini dirancang untuk membantu peserta didik memahami cara menyimpan dan mengelola kumpulan data menggunakan array, merepresentasikan data yang lebih kompleks melalui object, serta memanipulasi elemen pada halaman web menggunakan DOM. Melalui pembelajaran ini, peserta didik diharapkan mampu mengembangkan kemampuan berpikir logis dan menerapkan konsep JavaScript untuk membangun aplikasi web yang lebih dinamis, interaktif, dan sesuai dengan kebutuhan pengguna.
