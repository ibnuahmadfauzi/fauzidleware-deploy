#10. Elemen Multimedia

Website modern tidak hanya berisi teks, gambar, dan tabel. Tetapi juga dapat menampilkna berbagai jenis media seperti audio dan video. Kehadiran multimedia membuat website menjadi lebih menarik, interaktif, dan informatif. Contoh penggunaan multimedia pada website antara lain:

  • Musik pada website hiburan
  • Video pembelajaran
  • Podcast
  • Video profil perusahaan
  • Dokumentasi kegiatan
  • Tutorial interaktif

Sebelum HTML5 diperkenalkan, pemutaran audio dan video pada website memerlukan plugin tambahan seperti Adobe Flash. Namun, sejak hadirnya HTML5, browser telah menyediakan dukungan bawaan untuk menampilkan audio dan video menggunakan elemen <audio> dan <video>.



10.1. Elemen Multimedia dalam HTML

Multimedia adalah segala bentuk media yang menggabungkan suara, gambar, animasi, maupun video dalam sebuah halaman web. HTML menyediakan dua elemen utama untuk multimedia, yaitu:

Elemen Fungsi
<audio> Menampilkan file audio
<video> Menampilkan file video

Kedua elemen tersebut merupakan bagian dari standar HTML5 dan didukung oleh hampir semua browser modern.



10.2. Menampilkan Audio

Untuk menampilkan audio digunakan elemen <audio>. Berikut contohnya:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

Hasilnya:

Gambar 10.1 : contoh menampilkan audio


Pada contoh di atas:

  • <audio> digunakan untuk membuat pemutar audio
  • controls menampilkan tombol pemutar
  • <source> menentukan lokasi file audio
  • type menjelaskan format file audio

Browser mendukung beberapa format audio:

Format MIME Type
MP3 audio/mpeg
OGG audio/ogg
WAV audio/wav


10.3. Menampilkan VIdeo

Untuk menampilkan video digunakan elemen <video>. Berikut contohnya:

<video controls>
    <source src="video.mp4" type="video/mp4">
</video>

Gambar 10.2 : contoh menampilkan video


Beberapa format video:

Format MIME Type
MP4 video/mp4
WebM video/webm
OGG video/ogg

Format MP4 merupakan format yang paling umum digunaka karena didukung oleh hampir semua browser modern.
