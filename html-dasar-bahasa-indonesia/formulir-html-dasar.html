#8. Formulir HTML Dasar

Website modern tidak hanya digunakan untuk menampilkan informasi, tetapi juga sebagai media interaksi antara pengguna dan sistem. Salah satu bentuk interaksi tersebut adalah melalui formulir (form). Formulir digunakan untuk mengumpulkan data dari pengguna, seperti:

  • Login akun
  • Registrasi pengguna
  • Formulir kontak
  • Pemesanan produk
  • Pengisian biodata
  • Pendaftaran kegiatan

Dalam HTML, formulir dibuat menggunakan elemen <form> yang berisi berbagai komponen input.



8.1. Pengertian Formulir

Formulir adalah sekumpula elemen HTML yang digunakan untuk menerima masukan (input) dari pengguna. Data yang dimasukkan pengguna nantinya dapat diproses oleh aplikasi web menggunkan bahasa pemrograman seperti PHP, Python, Java, atau Node.js. Pada bab ini, kita hanya akan mempelajari cara membuat struktur formulir menggunakan HTML tanpa membahas proses pengolahan datanya.



8.2. Tag <form>

Elemen <form> merupakan wadah utama untuk semua komponen formulir. Contohnya:

<form>

    <input type="text">

</form>

Seluruh elemen input sebaiknya ditempatkan di dalam elemen <form>.



8.3. Atribut Penting pada <form>

Dua atribut yang sering digunakan adalah:

Atribut Fungsi
action=" " Menentukan tujuan pengiriman data
method=" " Menentukan metode pengiriman data
<form action="proses.php" method="post">

</form>

Pada tutorial HTML dasar ini, atribut tersebut cukup dikenalkan terlebih dahulu. Pembahasan lebih lanjut akan dipelajari saat mempelajari bahasa pemrograman web.



8.4. Input Teks

Input teks digunakan untuk menerima data berupa tulisan pendek, seperti nama atau nama kota. Berikut contohnya:

<label>Nama Lengkap</label><br>

<input type="text">


8.5. Menambahkan Placeholder

Placeholder merupakan teks penunjuk yang muncul sebelum pengguna mengetik. Contohnya:

<label>Nama Lengkap</label><br>

<input type="text" placeholder="Masukkan nama lengkap">


8.6. Input Password

Digunakan untuk memasukkan kata sandi. Karakter yang diketik akan disamarkan. Contohnya

<label>Password</label><br>

<input type="password" placeholder="Masukkan password">


8.7. Input Email

Digunakan untuk menerima alamat email. Browser dapat membantu memeriksa apakah format email sudah benar. Contohnya:

<label>Email</label><br>

<input type="email" placeholder="nama@email.com">


8.8. Input Angka

Digunakan untuk menerima data berupa angka. Bisa ditambah tambahan atribut min=" " dan max=" " untuk membatasi nilai. Berikut contohnya:

<label>Umur</label><br>

<input type="number" min="1" max="100">

Pada contoh tersebut pengguna hanya dapat memasukkan antara angka 1 hingga 100.



8.9. Penerapan Formulir Sederhana

Setelah mempelajari beberapa elemen formulir di atas, sekarang akan langsung kita implementasikan saja ke halaman web belajar-html dan lakukan modifikasi pada file kontak.html yaitu halaman Kontak.

<!doctype html>
<html>
  <head>
    <title>Kontak</title>
  </head>
  <body>
    <h1>Kontak</h1>

    <a href="../index.html">Beranda</a> | <a href="profil.html">Profil</a> |
    <a href="kontak.html">Kontak</a>

    <hr />

    <p>Email: admin@example.com</p>

    <h1>Hubungi</h1>

    <form action="#" method="post">
      <p>
        <label for="nama">Nama</label><br />
        <input
          type="text"
          id="nama"
          name="nama"
          placeholder="Masukkan nama Anda"
          required
        />
      </p>

      <p>
        <label for="telepon">Nomor HP</label><br />
        <input
          type="tel"
          id="telepon"
          name="telepon"
          placeholder="08xxxxxxxxxx"
          required
        />
      </p>

      <p>
        <label for="email">Email</label><br />
        <input
          type="email"
          id="email"
          name="email"
          placeholder="nama@email.com"
          required
        />
      </p>

      <p>
        <label for="subject">Subject</label><br />
        <input
          type="text"
          id="subject"
          name="subject"
          placeholder="Masukkan subject"
          required
        />
      </p>

      <p>
        <label for="pesan">Pesan</label><br />
        <textarea
          id="pesan"
          name="pesan"
          rows="5"
          cols="40"
          placeholder="Tuliskan pesan Anda..."
          required
        ></textarea>
      </p>

      <p>
        <input type="submit" value="Kirim" />
        <input type="reset" value="Reset" />
      </p>
    </form>
  </body>
</html>

Gambar 8.1 : penggunaan tabel di halaman kontak


Dari contoh di atas kita belajar beberapa elemen baru dan atribut baru. Elemen <textarea> digunakan untuk input teks dalam ukuran yang besar, contohnya untuk input pesan dari pengguna. Di dalam elemen <textarea> ada atribut rows=" " dan cols=" " untuk mengatur besar dari elemen tersebut. Lalu adalagi atribut require yang digunakan agar input tersebut tidak boleh kosong.

<input type="submit"> digunakan untuk membuat tombol yang difungsikan mengirimkan data dari semua inputnya. dan <input type="reset"> digunakan untuk membuat tombol yang berfungsi mengulang semua masukan dari pengguna.

Atribut lain seperti name=" " dan value=" " akan dibahas pada materi bahasa pemrograman web yang akan datang. Jika mau mendapatkan kode di atas bisa unduh pada link berikut : 8-1-form
