#12. Persiapan Belajar CSS

Selamat!!, kita telah mempelajari berbagai materi HTML, mulai dari struktur dasar sampai belajar tentang semantic HTML. Jika analoginya sebuah rumah, maka HTML adalah kerangka bangunan. HTML menentukan letak pintu, jendela, ruang tamu, kamar, dan bagian lainnya.

Namun, sebuah rumah tentu akan terlihat kurang menarik apabila hanya berupa kerangka tanpa cat, lantai, plafon, maupun dekorasi. Agar tampilannya menjadi indah dan nyaman dipandang, diperlukan proses desain. Dalam pengembangan website, proses mempercantik tamilan tersebut dilakukan menggunakan CSS (Cascading Styling Sheets).



12.1. Apa Itu CSS ?

CSS adalah bahasa stylesheet yang digunakan untuk mengatur tamppilan dan tata letak halaman website. Dengan CSS, kita dapat mengatur berbagai aspek tampilan, seperti:

  • Warna teks
  • Warna latar belakang
  • Jenis huruf
  • Ukuran huruf
  • Jarak antar elemen
  • Posisi elemen
  • Ukuran gambar
  • Tata letak halaman
  • Animasi sederhana
  • Tampilan website pada berbagai ukuran layar

HTML bertugas menyusun struktur halaman, sedangkan CSS bertugas mengatur bagaimana halaman tersebut ditampilkan kepada pengguna.



12.2. Mengapa Harus Belajar CSS ?

Jika hanya menggunakan HTML, halaman website tetap dapat dibuka oleh browser. Namun tampilannya masih sangat sederhana. Sebagai contoh:

  • Semua teks menggunakan jenis huruf bawaan browser
  • Warna halaman masih standar
  • Tata letak belum rapi
  • Menu navigasi belum menarik
  • Gambar belum diatur ukurannya
  • Tabel dan formulir masih terlihat sederhana

CSS digunakan untuk mengatasi keterbatasan tersebut sehingga websie menjadi lebih menarik dan nyaman digunakan.



12.3. Perbedaan HTML dan CSS
HTML CSS
Menyusun struktur halaman Mengatur tampilan halaman
Membuat heading, paragraf, tabel, gambar, formulir Mengatur warna, ukuran, posisi, dan tata letak
Menentukan isi website Menentukan desain website
Wajib dimiliki setiap halaman web Digunakan untuk mempercantik halaman

HTML dan CSS saling melengkapi. Keduanya hampir selalu digunakan secara bersamaan dalam pengembangan website modern.



12.4. Website HTML Semantic

Untuk menutup materi kita buat satu halaman web HTML dengan susunan semantic yang akan digunakan untuk praktek materi CSS yang akan datang.

Berikut susunan foldernya:

belajar-html/
│
├── index.html
│
├── pages/
│   ├── profil.html
│   └── kontak.html


Halaman Beranda - File index.html

<!doctype html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <title>Beranda - Belajar HTML</title>
  </head>
  <body>
    <header>
      <h1>Belajar HTML Dasar</h1>
      <p>Website sederhana menggunakan Semantic HTML.</p>
    </header>

    <nav>
      <a href="index.html">Beranda</a> |
      <a href="pages/profil.html">Profil</a> |
      <a href="pages/kontak.html">Kontak</a>
    </nav>

    <main>
      <section>
        <h2>Selamat Datang</h2>

        <article>
          <p>
            Selamat datang di website sederhana yang dibuat menggunakan HTML.
            Website ini terdiri dari tiga halaman yang saling terhubung melalui
            menu navigasi.
          </p>

          <p>
            Seluruh struktur halaman menggunakan Semantic HTML agar kode lebih
            rapi dan mudah dipahami.
          </p>
        </article>
      </section>

      <aside>
        <h3>Menu Cepat</h3>

        <ul>
          <li>Beranda</li>
          <li>Profil</li>
          <li>Kontak</li>
        </ul>
      </aside>
    </main>

    <footer>
      <p>&copy; 2026 Belajar HTML Dasar</p>
    </footer>
  </body>
</html>


Halaman Profil - File profil.html

<!doctype html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <title>Profil - Belajar HTML</title>
  </head>
  <body>
    <header>
      <h1>Profil Website</h1>
      <p>Mengenal lebih dekat website ini.</p>
    </header>

    <nav>
      <a href="../index.html">Beranda</a> | <a href="profil.html">Profil</a> |
      <a href="kontak.html">Kontak</a>
    </nav>

    <main>
      <section>
        <article>
          <h2>Tentang Website</h2>

          <p>
            Website ini dibuat sebagai media pembelajaran HTML dasar. Seluruh
            contoh yang digunakan bertujuan membantu pemula memahami struktur
            dokumen HTML.
          </p>
        </article>

        <article>
          <h2>Tujuan</h2>

          <ul>
            <li>Belajar struktur HTML.</li>
            <li>Mengenal Semantic HTML.</li>
            <li>Membuat website sederhana.</li>
          </ul>
        </article>
      </section>

      <aside>
        <h3>Informasi</h3>

        <p>
          Website ini belum menggunakan CSS sehingga tampilannya masih mengikuti
          tampilan bawaan browser.
        </p>
      </aside>
    </main>

    <footer>
      <p>&copy; 2026 Belajar HTML Dasar</p>
    </footer>
  </body>
</html>

        <p>Telepon: 0812-3456-7890</p>
      </aside>
    </main>

    <footer>

    </footer>
