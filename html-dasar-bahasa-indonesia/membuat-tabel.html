#7. Membuat Tabel

Tabel merupakan salah satu elemen HTML yang digunakan untuk menyajikan data dalam bentuk baris dan kolom. Penyajian data menggunakan tabel membuat informasi menjadi lebih terstruktur, mudah dibaca, dan mudah dibandingkan. Beberapa contoh penggunaan tabel pada website antara lain:

  • Data siswa
  • Jadwal pelajaran
  • Daftar harga
  • Laporan penjualan
  • Data karyawan
  • Nilai ujian

HTML menyediakan berbagai elemen khusus yang digunakan untuk membuat tabel secara tersturktur.



7.1. Pengertian Tabel

Tabel adalah sekumpulan data yang disusun dalam bentuk baris (row) dan kolom (column). Setiap perpotongan antar baris dan kolom disebut sebagai sel (cell). Contoh tampilan tabel:

No Nama Kelas
1 Andi VII A
2 Budi VII B
3 Citra VII A

Pada HTML, tabel dibuat menggunakan beberapa elemen yang saling melengkapi.



7.2. Tag <table>

Elemen <table> digunakan sebagai wadah utama seluruh isi tabel. Semua bagian tabel seperti baris, kolom, dan judul harus berada di dalam elemen <table>. Berikut contoh dasar dari penggunaan elemen <table>:

<table>
    <tr>
        <td>HTML</td>
    </tr>
</table>

Pada contoh di atas:

  • <table> membuat tabel
  • <tr> membuat satu baris
  • <td> membuat satu sel


7.3. Baris Tabel (<tr>)

Elemen <tr> merupakan singkatan dari Table Row. Elemen ini digunakan untuk membuat satu baris pada tabel. Contohnya:

<table>
    <tr>
        <td>Andi</td>
        <td>VII A</td>
    </tr>
</table>

Gambar 7.1 : contoh tampilan tabel


perhatikan bahwa secara default elemen <table> tidak membawa garis atau border. Jadi untuk menambahkannya bisa pakai atribut border=" ". Contohnya:

<table border="1">
  <tr>
    <td>Andi</td>
    <td>VII A</td>
  </tr>
</table>

Jika ingin mengatur jarak antar baris pakai atribut cellspacing=" ", jika ingin mengatur jarak baris ke konten pakai atribut cellpadding=" "

<table border="1" cellspacing="0" cellpadding="10">
  <tr>
    <td>Andi</td>
    <td>VII A</td>
  </tr>
</table>

Sekarang coba kita buat beberapa baris lagi:

<table border="1" cellspacing="0" cellpadding="10">
  <tr>
    <td>1</td>
    <td>Andi</td>
    <td>VII A</td>
  </tr>

  <tr>
    <td>2</td>
    <td>Budi</td>
    <td>VII B</td>
  </tr>
</table>

Gambar 7.2 : contoh tampilan beberapa baris tabel




7.4. Kolom Tabel (<td>)

Elemen <td> merupakan singkatan dari Table Data. Digunakan untuk membuat isi setiap sel tabel. Contohnya sudah kita coba sebelumnya.



7.5. Judul Kolom (<th>)

Elemen <th> merupakan singkatan dari Table Header. Digunakan untuk membuat judul kolom. Browser biasanya menampilkan isi <th> dengan huruf tebal. Contohnya:

<table border="1" cellspacing="0" cellpadding="10">
  <tr>
    <th>No.</th>
    <th>Nama</th>
    <th>Kelas</th>
  </tr>

  <tr>
    <td>1</td>
    <td>Andi</td>
    <td>VII A</td>
  </tr>

  <tr>
    <td>2</td>
    <td>Budi</td>
    <td>VII B</td>
  </tr>
</table>

Hasilnya:

Gambar 7.3 : contoh tampilan judul tabel


Selain membuat tampilan lebih jelas, elemen <th> juga memberikan makna semantik sehingga browser dapat mengenali bahwa sel tersebut adalah judul kolom.



7.6. Penggabungan Kolom (colspan)

Kadang-kadang sebuah sel perlu melebar hingga mencakup beberapa kolom. Untuk itu digunakan atribut colspan. Berikut contoh penggunaannya:

<table border="1" cellspacing="0" cellpadding="10">
  <tr>
    <th>No.</th>
    <th>Nama</th>
    <th>Kelas</th>
  </tr>

  <tr>
    <td>1</td>
    <td>Andi</td>
    <td>VII A</td>
  </tr>

  <tr>
    <td>2</td>
    <td>Budi</td>
    <td>VII B</td>
  </tr>

  <tr>
    <td colspan="3">Data Siswa Aktif</td>
  </tr>
</table>

Hasilnya:

Gambar 7.4 : contoh penggabungan kolom


Biasanya atribut colspan=" " digunakan untuk judul tabel, ringkasan, pengelompokan nilai, total nilai, ataupun pengelompokan data.



7.7. Penggabungan Baris (rowspan)

Selain kolom, HTML juga memungkinkan penggabungan beberapa baris menggunakan atribut rowspan=" ". Berikut contoh penggunannya:

<table border="1" cellspacing="0" cellpadding="10">
  <tr>
    <th>No.</th>
    <th>Nama</th>
    <th>Kelas</th>
    <th rowspan="4">Kelas VII</th>
  </tr>

  <tr>
    <td>1</td>
    <td>Andi</td>
    <td>VII A</td>
  </tr>

  <tr>
    <td>2</td>
    <td>Budi</td>
    <td>VII B</td>
  </tr>

  <tr>
    <td colspan="3">Data Siswa Aktif</td>
  </tr>
</table>

Dari contoh di atas, atribut rowspan="4" akan mengambil 4 baris kebawah untuk digabungkan.

7.8. Header, Body, dan Footer Tabel

Agar struktur tabel lebih rapi, HTML menyediakan tiga elemen tambahan:

Elemen Fungsi
<thead> Bagian kepala tabel
<tbody> Bagian isi tabel
<tfoot> Bagian akhir tabel
