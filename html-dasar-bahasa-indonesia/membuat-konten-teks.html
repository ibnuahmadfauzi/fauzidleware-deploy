#3. Membuat Konten Teks

Sebagian besar halaman web berisi informasi dalam bentuk teks. Oleh karena itu, HTML menyediakan berbagai elemen yang dapat digunakan untuk menyusun dan memformat teks agar lebih mudah dibaca oleh pengunjung.


3.1. Heading (<h1> sampai <h6>)

Heading digunakan untuk membuat judul dan subjudul pada halaman web. HTML menyediakan enam tingkat heading, yaitu:

  • <h1>
  • <h2>
  • <h3>
  • <h4>
  • <h5>
  • <h6>

Semakin kecil angka pada heading, semakin besar tingkat kepentingannya. Contohnya:

<h1>Belajar HTML</h1>
<h2>Pengenalan HTML</h2>
<h3>Struktur Dasar HTML</h3>
<h4>Tag HTML</h4>
<h5>Atribut HTML</h5>
<h6>Latihan HTML</h6>

Hindari langsung menggunakan <h4> tanpa adanya <h2> atau <h3> sebelumnya.



3.2. Paragraf (<p>)

Paragraf digunaka untuk menampilkan blok teks. Contohnya:

<p>HTML adalah bahasa markup yang digunakan untuk membuat halaman web.</p>

<p>HTML menjadi dasar dari hampir semua website yang ada di internet.</p>

Browser akan memberikan jarak otomatis antar paragraf.

Mengapa menggunakan <p> ?

Tanpa elemen paragraf:

HTML adalah bahasa markup.
HTML digunakan untuk membuat website.

Browser akan menampilkan teks secara berurutan tanpa struktur yang jelas. Jadi dengan paragraf tampilan menjadi lebih rapi dan mudah dibaca.



3.3. Baris Baru (<br>)

Tag <br> digunakan untuk membuat baris baru tanpa membuat paragraf baru. Contohnya:

<p>
  Nama: Ahmad<br />
  Alamat: Blitar<br />
  Pekerjaan: Web Developer
</p>

Kapan Menggunakan <br> ?

Biasanya digunakan untuk:

  • Alamat
  • Puisi
  • Lirik lagu
  • Data idetitas

Hindari menggunakan <br> untuk mengatur jarak antar paragraf. Dan tag ini adalah single tag jadi tidak punya penutup, bisa ditulis <br> atau <br />



3.4. Garis Horizontal (<hr>)

Tag <hr> digunakan untuk membuat garis pemisah antar bagian konten. Contohnya:

<h1>Tutorial HTML</h1>

<p>Materi dasar HTML.</p>

<hr />

<p>Materi lanjutan HTML.</p>

Hasilnya akan menampilkan garis horizontal di area antara kedua paragraf.

Secara umum tag <hr> berfungsi untuk:

  • Memisahkan topik pembahasan
  • Menunjukkan pergantian bagian konten
  • Membantu pembaca memahami struktur artikel


3.5. Penekanan Teks (<strong> dan <em>)

HTML menyediakan elemen khusus untuk memberikan penekanan pada teks.

1. Strong (<strong>)

Digunakan untuk menunjukkan bahwa suatu teks memiliki tingkat kepentingan yang tinggi. Contohnya:

<p>
HTML adalah <strong>bahasa markup</strong> untuk membuat website.
</p>

Biasanya browser menampilkan teks menjadi tebal.

2. Emphasis (<em>)

Digunakan untuk memberikan penekanan pada teks. Contohnya:

<p>
Belajar HTML akan menjadi lebih mudah jika dilakukan secara <em>konsisten</em>.
</p>

Biasanya browser menampilkan teks menjadi miring.

3. Perbedaan Strong dan Em

Elemen Fungsi
<strong> Menunjukkan tingkat kepentingan
<em> Memberikan penekanan


3.6. Teks Tebal dan Mirig

Selain <strong> dan <em>, HTML juga menyediakan elemen:

  • <b> (bold)
  • <i> (italic)

Contohnya:

<p><b>HTML</b> adalah bahasa markup.</p>
<p><i>Website</i> merupakan kumpulan halaman web.</p>

Perbedaan dengan elemen sebelumnya

Elemen Makna
<strong> Penting
<b> hanya tampilan tebal
<em> Penekanan
<i> Hanya tampilan miring

Dalam HTML modern, penggunaan <strong> dan <em> lebih disarankan karena memiliki makna semantik.



3.7. Superscript dan Subscript

HTML menyediakan elemen untuk menampilkan teks di atas atau di bawah garis normal.

1. Superscript (<sup>)

Digunakan untuk pangkat atau catatan kaki. Contohnya:

<p>
2<sup>3</sup> = 8
</p>

2. Subscript (<sub>)

Digunakan untuk rumus kimia atau notasi tertentu. Contohnya:

<p>
H<sub>2</sub>O
</p>


3.8. Menampilkan Kutipan

HTML menyediakan elemen untuk menampilkan kutipan. Contohnya:

<p>
Menurut Tim Berners-Lee, <q>Web is for everyone.</q>
</p>
<blockquote>
HTML adalah fondasi utama dalam pengembangan website modern.
</blockquote>

<1> digunakan untuk kutipan pendek dan <blockquote> digunakan untuk kutipan yang lebih panjang. Biasanya browser akan menambahkan tanda kutip atau indentasi khusus otomatis pada kutipan.

Kapan menggunakan kutipan ?

Kutipan sering digunakan pada:

  • Artikel
  • Blog
  • Penelitian
  • Dokumentasi
  • Berita


3.9. Praktik Membuat Artikel Sederhana

Mari menggabungkan seluruh materi yang telah dipelajari. Buat file baru dalam folder belajar-html/pages/ dengan nama artikel.html. Lalu isi dengan kode berikut:

<!doctype html>
<html>
  <head>
    <title>Artikel HTML</title>
  </head>
  <body>
    <h1>Belajar HTML untuk Pemula</h1>

    <p>
      HTML merupakan bahasa markup yang digunakan untuk membuat halaman web.
    </p>

    <hr />

    <h2>Mengapa Belajar HTML?</h2>

    <p>HTML adalah fondasi utama dalam pengembangan website modern.</p>

    <p>
      Dengan memahami <strong>HTML</strong>, Anda dapat membuat berbagai jenis
      halaman web dengan lebih mudah.
    </p>

    <p>
      Konsistensi adalah kunci untuk menjadi seorang
      <em>web developer</em>.
    </p>

    <h2>Contoh Rumus</h2>

    <p>H<sub>2</sub>O adalah rumus kimia air.</p>

    <p>2<sup>4</sup> = 16</p>

    <h2>Kutipan</h2>

    <blockquote>
      Belajar sedikit setiap hari lebih baik daripada belajar banyak tetapi
      hanya sekali.
    </blockquote>
  </body>
</html>
