#1. Pengenalan HTML

1.1. Apa Itu HTML ?

HTML merupakan singkatan dari HyperText Markup Language, yaitu bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman web. HTML bukan bahasa pemrograman, melainkan bahasa yang digunakan untuk memberi tanda (markup) pada berbagai bagian dokumen sehingga browser dapat menampilkan informasi dengan benar.

Ketika Anda membuka sebuah website, browser seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge akan membaca kode HTML dan menerjemahkannya menjadi tampilan yang dapat dilihat pengguna.

Sebagai contoh, sebuah judul artikel dapat ditandai menggunakan tag heading, sedangkan isi artikel dapat ditandai menggunakan tag paragraf.

Contoh sederhana HTML:

<h1>Belajar HTML</h1>
<p>HTML adalah bahasa markup untuk membuat website.</p>

Mengapa HTML Penting ?

Tanpa HTML, sebuah website tidak akan memiliki struktur yang jelas. HTML berfungsi sebagai kerangka dasar yang nantinya dapat dipercantik menggunakan CSS dan ditambahkan interaktivitas menggunakan JavaScript.

Hubungan ketiganya dapat diibaratkan sebagai berikut:

Teknologi Fungsi
HTML Struktur bangunan
CSS Desain dan dekorasi
JavaScript Interaksi dan fungsi


1.2. Sejarah Perkembangan HTML

HTML pertama kali dikembangkan oleh seorang ilmuwan komputer bernama Tim Berners-Lee pada tahun 1991.

Tujuan awal HTML adalah memudahkan para peneliti untuk berbagi dokumen melalui jaringan internet yang kemudian berkembang menjadi World Wide Web (WWW).

Perkembangan HTML dari waktu ke waktu:

Tahun Versi
1991 HTML pertama
1995 HTML 2.0
1997 HTML 3.2
1999 HTML 4.01
2014 HTML5
Sekarang HTML Living Standard

HTML5 merupakan versi HTML yang paling banyak digunakan saat ini karena menyediakan berbagai fitur modern, seperti:

  • Audio
  • Video
  • Semantic HTML
  • Form yang lebih lengkap
  • Dukungan perangkat mobile yang lebih baik

Meskipun HTML terus berkembang, prinsip dasar penggunanya tetap sama.



1.3. Fungsi HTML dalam Pembuatan Website

HTML memiliki berbagai fungsi penting dalam pengembangan website.

1. Membuat Struktur Halaman

HTML digunakan untuk menentukan bagian-bagian halaman, seperti:

  • Judul
  • Paragraf
  • Gambar
  • Tabel
  • Formulir
  • Navigasi

Contoh:

<h1>Website Sekolah</h1>
<p>Selamat datang di website sekolah kami.</p>

2. Menampilkan Konten

HTML membantu browser menampilkan berbagai jenis konten, seperti:

  • Teks
  • Gambar
  • Audio
  • Video
  • Dokumen

3. Menghubungkan Halaman

HTML memungkinka pembuatan tautan antar halaman menggunakan tag <a>.

Contoh:

<a href="profil.html">Profil Sekolah</a>

4. Membuat Formulir

HTML dapat digunakan untuk membuat formulir pendaftaran, login, kontak, dan sebagainya.

Contoh:

<input type="text">

5. Membantu SEO dan Aksesibilitas

Struktur HTML yang baik membantu Mesin pencari memahami isi website



1.4. Struktur Dasar Dokumen HTML

Setiap halaman HTML memiliki struktur dasar sebagai berikut:

<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama</title>
</head>
<body>

    <h1>Hello World</h1>

</body>
</html>

Penjelasan:

  • <!DOCTYPE html> menyatakan bahwa dokumen menggunakan standar HTML5.
  • <html> elemen utama yang membungkus seluruh dokumen.
  • <head> berisi informasi tentang halaman. Contohnya judul halaman, metadata, dan informasi SEO.
  • <title> menentukan judul tab browser.
  • <body> berisi seluruh konten yang ditampilkan kepada pengguna.


1.5. Mengenal Elemen, Tag, dan Atribut

Saat belajar HTML, anda akan sering mendengar istilah elemen, tag, dan atribut.


1. Tag

Tag merupakan penanda yang digunakan HTML. Cotohnya <h1> dan </h1>.

<tag pembuka> dan </tag penutup>


2. Elemen

Elemen adalah keseluruhan bagian yang terdiri dari tag pembuka, isi atau konten, dan tag penutup.

<tag pembuka> ... konten ... </tag penutup>

Contohnya:

 <h1> Belajar HTML </h1>

Elemen tersebut terdiri dari:

Tag pembuka <h1>
Isi / konten Belajar HTML
Tag penutup </h1>

3. Atribut

Atribut digunakan untuk memberi informasi tambahan pada elemen.

Contoh:

<a href="profil.html">Halaman Profil</a>

Pada contoh tersebut href adalah atribut dan profil.html adalah nilai atribut. Dalam 1 elemen bisa berisi lebih dari satu atribut, seperti berikut ini:

<img src="gambar.jpg" alt="Pemandangan">



1.6. Aturan Penulisan Kode HTML

Agar kode HTML mudah dibaca dan dipelihara, ikuti beberapa aturan berikut.


1. Gunakan huruf kecil

Disarankan menggunakan <h1> ... </h1> daripada <H1> ... </H1>


2. Tutup tag yang memiliki pasangan

<h1> Judul </h1> Benar
<h1> Judul Salah
<img src="logo.png"> Benar
memang tidak memiliki tag penutup atau single tag
<img src="logo.png" /> Benar
versi lain single tag lebih rapi

3. Gunakan indentasi

Indentasi membuat kode lebih rapi walaupun tanpa indentasi kode HTML tetap bisa berjalan. Berikut contoh yang menggunakan indentasi:

<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama</title>
</head>
<body>

    <h1>Belajar HTML</h1>
    <p>Belajar html itu menyenangkan</p>

</body>
</html>

Berikut kode HTML tanpa indentasi:

<!DOCTYPE html>
<html>
<head>
<title>Website Pertama</title>
</head>
<body>
<h1>Belajar HTML</h1>
<p>Belajar html itu menyenangkan</p>
</body>
</html>

4. Gunakan nama file yang baik

Contohnya: index.html, profil.html, kontak.html

Hindari: File Baru.html, Profil Saya Final Banget.html


5. Berikan komentar pada kode jika diperlukan

Contoh:

 
<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama</title>
</head>
<body>

    <!-- judul heading 1 -->
    <h1>Belajar HTML</h1>

    <!-- paragraf pertama -->
    <p>Belajar html itu menyenangkan</p>

</body>
</html>

<!-- ini adalah komentar -->

Elemen komentar tidak akan tampil di browser.
