#9. Formulir HTML Lanjutan

Pada materi sebelumnya, kita telah mempelajari cara membuat formulir sederhana menggunakan berbagai jenis input seperti teks, email, password, angka, tanggal, telepon, serta textarea.

Namun, formulir pada sebuah website umumnya tidak hanya terdiri atas input teks saja. Banyak data yang lebih tepat diinputkan menggunakan pilihan tertentu agar pengguna lebih mudah mengisi formulir dan meminimalkan kesalahan. Sebagai contoh:

  • Memilih jenis kelamin
  • Memilih agama
  • Memilih hobi
  • Memilih jurusan
  • Memilih kota
  • Memilih tingkat pendidikan

HTML menyediakan beberapa elemen khusus untuk kebutuhan tersebut, diantaranya Radio Button, Checkbox, Select Option, dan Optgroup.



9.1. Radio Button

Radio Button digunakan ketika pengguna hanya boleh memilih satu pilihan dari beberapa pilihan yang tersedia. Contoh penggunaan Radio Button:

  • Jenis Kelamin
  • Status Pernikahan
  • Metode Pembayaran
  • Pilihan Kelas

Contoh sederhana:

<label>
    <input
        type="radio"
        name="jk"
        value="L"
        checked>
    Laki-laki
</label>

<br>

<label>
    <input
        type="radio"
        name="jk"
        value="P">
    Perempuan
</label>

Gambar 9.1 : contoh radio button


Mengapa perlu atribut name=" " ?. Radio Button yang memiliki nilai name=" " yang sama hanya dapat dipilih salah satu pilihan. Jika atribut name=" " nilainya berbeda, maka semua Radio Button dapat dipilih secara bersamaan, sehingga fungsi Radio Button menjadi tidak tepat.

Atribut value=" " digunakan sebagai nilai pilihan Radio Button yang dikirim ketika formulir diproses.

Atribut checked akan membuat Radio Button tersebut terpilih otomatis pada awal digunakan (bisa digunakan bisa tidak). Jika tidak digunakan maka ketika awal formulir dibuka tidak ada Radio Button yang dipilih.



9.2. Checkbox

Checkbox digunakan ketika pengguna boleh memilih lebih dari satu pilihan. Contoh penggunaannya:

  • Hobi
  • Bahasa Pemrograma
  • Makanan Favorit
  • Mata Pelajaran Favorit

Berikut contoh struktur penggunannya:

<label>
    <input
        type="checkbox"
        name="hobi"
        value="Membaca">
    Membaca
</label>

<br>

<label>
    <input
        type="checkbox"
        name="hobi"
        value="Olahraga"
        checked>
    Olahraga
</label>

<br>

<label>
    <input
        type="checkbox"
        name="hobi"
        value="Musik">
    Musik
</label>

Gambar 9.2 : contoh checkbox




9.3. Select Option

Jika pilihan sangat banyak, penggunaan Radio Button akan membuat halaman menjadi panjang. Solusinya adalah menggunakan elemen <select>. Contoh struktur dasar penggunaannya:

<select name="programming-language">
    <option value="php">PHP</option>
    <option value="html" selected>HTML</option>
    <option value="css">CSS</option>
    <option value="js">JavaScript</option>
    <option value="cpp">C++</option>
    <option value="java">Java</option>
</select>

9.4. Optgroup

Jika daftar pilihan sangat banyak, kita dapat mengelompokkannya menggunakan elemen <optgroup>. Hal ini membuat daftar pilihan menjadi lebih rapi dan mudah dipahami.

Contoh formulir sederhana gabungan dari materi di atas:

<!doctype html>
<html>
  <head>
    <title>Form Biodata</title>
  </head>
  <body>
    <h1>Form Biodata</h1>

    <form>
      <p>
        <label>Jenis Kelamin</label><br />

        <label>
          <input type="radio" name="jk" value="L" />
          Laki-laki
        </label>

        <label>
          <input type="radio" name="jk" value="P" />
          Perempuan
        </label>
      </p>

      <p>
        <label>Hobi</label><br />

        <label>
          <input type="checkbox" name="hobi" value="Membaca" />
          Membaca
        </label>

        <br />

        <label>
          <input type="checkbox" name="hobi" value="Olahraga" />
          Olahraga
        </label>

        <br />

        <label>
          <input type="checkbox" name="hobi" value="Musik" />
          Musik
        </label>
      </p>

      <p>
        <label>Bahasa Pemrograman Favorit</label><br />

        <select name="bahasa">
          <optgroup label="Front-End">
            <option>HTML</option>
            <option>CSS</option>
            <option>JavaScript</option>
          </optgroup>

          <optgroup label="Back-End">
            <option>PHP</option>
            <option>Python</option>
            <option>Node.js</option>
          </optgroup>
        </select>
      </p>
    </form>
  </body>
</html>
