#4. Membuat Daftar (List)

Dalam sebuah website, sering kali kita perlu menampilkan bentuk daftar. Misalnya:

  • Menu navigasi website
  • Daftar materi pembelajaran
  • Daftar produk
  • Daftar fitur aplikasi
  • Langkah-langkah tutorial

HTML menyediakan beberapa jenis list yang dapat digunakan sesuai kebutuhan. Secara umum terdapat tiga jenis list dalam HTML:

  • Ordered List (<ol>)
  • Unordered List (<ul>)
  • Description List (<dl>)


4.1. Ordered List (<ol>)

Ordered list digunakan untuk membuat daftar yang memiliki urutan tertentu. Biasanya untuk:

  • Langkah-langkah tutorial
  • Prosedur kerja
  • Peringkat
  • Instruksi

Contoh struktur dan penulisan ordered list:

<ol>
  <li>Menyalakan komputer</li>
  <li>Membuka browser</li>
  <li>Membuka website</li>
</ol>

Gunakan ordered list untuk konten yang memiliki urutan item penting, artinya jika urutan berubah maka informasi juga dapat berubah.



4.2. Unordered List (<ul>)

Unordered list digunakan untuk membuat daftar yang tidak memiliki urutan tertentu, biasanya ditampilkan dengan simbol bullet. Contoh penulisan unordered list:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Hasilnya:

Gambar 4.2 : hasil unordered list


Gunakan unordered list ketika membuat urutan item tidak penting, dan posisi item tidak mempengaruhi makna daftar.



4.3. List Item (<li>)

Elemen <li> merupakan singkatan dari list item, digunakan untuk membuat setiap item pada daftar. Cara penggunaannya sudah kita coba pada materi ordered list dan unordered list.

<ol>
    <li>Bab 1</li>
    <li>Bab 2</li>
    <li>Bab 3</li>
</ol>
<ul>
    <li>Apel</li>
    <li>Jeruk</li>
    <li>Mangga</li>
</ul>

Elemen <li> harus berada di dalam <ol> ataupun <ul>.



4.4. Nested List

Nested list adalah daftar yang berada di dalam daftar lainnya. Teknik ini digunakan untuk membuat struktur bertingkat.

<ul>
  <li>
    Front-End
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>

  <li>
    Back-End
    <ul>
      <li>PHP</li>
      <li>Python</li>
      <li>Node.js</li>
    </ul>
  </li>
</ul>

Kapan nested list digunakan ?

Nested list sering digunakan untuk:

  • Struktur menu website
  • Daftar kategori
  • Materi pembelajaran
  • Struktur organisasi
  • Daftar produk dna subkategori


4.5. Description List (<dl>, <dt>, <dd>)

Description list digunaka untuk membuat daftar istilah dan penjelasannya. Jenis list ini berbeda dari ordered list maupun unordered list.

Elemen description list:

Elemen Fungsi
<dl> Description List
<dt> Description Term
<dd> Description Definition

Berikut struktur dan penggunaan description list:

<dl>

    <dt>HTML</dt>
    <dd>Bahasa markup untuk membuat halaman web.</dd>

    <dt>CSS</dt>
    <dd>Bahasa yang digunakan untuk mengatur tampilan website.</dd>

    <dt>JavaScript</dt>
    <dd>Bahasa pemrograman yang membuat website menjadi interaktif.</dd>

</dl>

Kapan menggunakan description list ?

Description list cocok digunakan untuk:

  • Glosarium istilah
  • Kamus
  • Profil pengguna
  • FAQ sederhana
  • Dokumentasi teknis
