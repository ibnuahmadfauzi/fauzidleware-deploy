#5. Membuat Hyperlink

Salah satu fitur terpenting dalam website adalah kemampuan berpindah dari satu halaman ke halaman lainnya. Kemampuan ini diwujudkan melalui hyperlink. Tanpa hyperlink, sebuah website hanya akan menjadi kumpulan halaman yang berdiri sendiri dan tidak saling terhubung.



5.1. Pengertian Hyperlink

Hyperlink adalah elemen yang digunakan untuk menghubungkan suatu halaman atau sumber daya dengan halaman atau sumber daya lainnya. Ketika pengguna mengklik hyperlink, browser akan membuka tujuan yang telah ditentukan. Contohnya:

<a href="profil.html">Profil Saya</a>

Ketika teks "Profil Saya" diklik, browser akan membuka file profil.html.



5.2. Tag Anchor <a>

Hyperlink dibuat menggunakan elemen Anchor <a>. Struktur dasar dari penggunaan elemen ini adalah seperti contoh berikut:

<a href="profil.html">Profil Saya</a>

5.3. Link ke Website Eksternal

Hyperlink juga dapat digunakan untuk membuka website lain. Contohnya:

<a href="https://www.google.com">Situs Google</a>

Untuk hyperlink ke website eksternal sebaiknya gunakan alamat lengkap dan menggunakan protokol https seperti contoh di atas.



5.4. Link ke Bagian Tertentu dalam Halaman

HTML memungkinkan pengguna berpindah ke bagian tertentu dalam halaman yang sama. Teknik ini disebut Anchor Link atau Bookmark Link.


Langkah 1 : Beri ID pada Elemen

<h2 id="bab1">Bab 1</h2>

Langkah 2 : Buat Link

<a href="#bab1">Pergi ke Bab 1</a>

Ketika diklik, browser akan menggulir halaman menuju elemen yang memiliki ID bab1.



5.5. Membuka Link pada Tab Baru

Secara default, hyperlink akan terbuka pada tab yang sama. Untuk membuka link pada tab baru gunakan atribut target="_blank". Seperti contoh berikut:

<a href="profil.html" target="_blank">Profil Saya</a>

Biasanya menggunakan tab baru untuk website eksternal, dokumen PDF, maupun halaman referensi. Sehingga pengunjung tidak meninggalkan halaman utama website.



5.6. Praktik Membuat Navigasi

Masih ingat pada materi-materi sebelumnya kita sudah membuat satu folder belajar-html dengan susunan file dan folder seperti berikut:

belajar-html/
│
├── index.html
│
├── pages/
│   ├── profil.html
│   └── kontak.html
│
└── images/

Sekarang kita buat masing-masing halaman punya konten dan hyperlink terhubung dengan halaman lain.

Isi file index.html

<!doctype html>
<html>
  <head>
    <title>Beranda</title>
  </head>
  <body>
    <h1>Website Saya</h1>

    <a href="index.html">Beranda</a> | <a href="pages/profil.html">Profil</a> |
    <a href="pages/kontak.html">Kontak</a>

    <hr />

    <p>Selamat datang di website saya.</p>
  </body>
</html>

Isi file profil.html

<!doctype html>
<html>
  <head>
    <title>Profil</title>
  </head>
  <body>
    <h1>Profil Saya</h1>

    <a href="../index.html">Beranda</a> | <a href="profil.html">Profil</a> |
    <a href="kontak.html">Kontak</a>

    <hr />

    <p>Ini adalah halaman profil.</p>
  </body>
</html>

Isi file kontak.html

<!doctype html>
<html>
  <head>
    <title>Kontak</title>
  </head>
  <body>
    <h1>Kontak</h1>

    <a href="../index.html">Beranda</a> | <a href="profil.html">Profil</a> |
    <a href="kontak.html">Kontak</a>

    <hr />

    <p>Email: admin@example.com</p>
  </body>
</html>

Gambar 5.2 : pembuatan hyperlink pada website



Dari contoh di atas, hyperlink digunakan untuk navigasi antar halaman. Perhatikan karena file profil.html dan kontak.html berada di dalam folder pages/, maka penulisan nilai atribut href=" " di file index.html harus menyertakan nama folder karena file yang dituju tidak berada di lokasi yang sama.

Begitu juga sebaliknya ketika berada di halaman Profil dan Kontak untuk akses ke halaman Beranda yaitu file index.html harus menyertakan penulisan ../ sebelum nama file, yang fungsinya untuk mundur satu folder keluar.
