#6. Menampilkan Gambar

Selain teks, gambar merupakan salah satu elemen penting dalam sebuah website. Gambar dapat digunakan untuk memperjelas informasi, mempercantik tampilan, memperkenalkan suatu produk, hingga menjadi media penyampaian pesan kepada pengunjung.

Website modern hampir selalu menggunakan gambar, baik berupa logo, foto, ilustrasi, ikon, maupun banner. HTML menyediakan elemen khusus untuk menampilkan gambar, yaitu elemen <img>



6.1. Pengertian Gambar dalam Website

Gambar pada website adalah file visual yang ditampilkan pada halaman web untuk mendukung penyampaian informasi kepada pengguna.

Beberapa contoh penggunaan gambar pada website antara lain:

  • Logo perusahaan
  • Foto profil
  • Foto produk
  • Banner promoso
  • Ilustrasi artikel
  • Ikon menu
  • Galeri foto

Dengan adanya gambar, informasi menjadi lebih menarik dan mudah dipahami oleh pengunjung.



6.2. Format Gambar yang Umum Digunakan

Beberapa format gambar yang sering digunakan dalam website adalah:

Format Keterangan
JPG/JPEG Cocok untuk foto
PNG Mendukung latar belakang transparan
GIF Mendukung animasi sederhana
SVG Gambar berbasis vektor
WebP Ukuran file lebih kecil dengan kualitas tinggi

Nantinya untuk contoh kita akan menggunakan gambar berformat JPG dan PNG.



6.3. Tag <img>

Untuk menampilkan gambar digunakan elemen <img>. Berbeda dengan sebagian besar elemen HTML, elemen <img> tidak memiliki tag penutup, sehingga cukup ditulis satu kali. Berikut struktur dasar penggunaan tag <img>.

<img src="gambar.jpg" alt="Deskripsi gambar">

Bagian Fungsi
<img> Menampilkan gambar
src=" " Lokasi gambar
alt=" " Deskripsi gambar

Contoh penggunaan:

<!doctype html>
<html>
  <head>
    <title>Coba Gambar</title>
  </head>
  <body>
    <img src="logo.png" alt="Gambar Logo" />
  </body>
</html>

6.4. Atribut src=" "

Atribut src=" " merupakan singkatan dari Source, yaitu lokasi atau alamat file gambar yang akan ditampilkan. Tanpa atribut src=" " browser tidak mengetahu gambar mana yang harus ditampilkan.

<img src="gambar.jpg" alt="Pemandangan">

Pada contoh di atas, src="gambar.jpg" menunjukkan lokasi file gambar. Browser akan mencari file gambar.jpg.

Contoh lain:

<img src="images/logo.png" alt="Logo">

Dengan begitu browser akan mencari file logo.png yang ada di dalam folder images/.



6.5. Atribut alt=" "

Atribut alt=" " merupakan singkatan dari Alternative Text. Atribut ini berfungsi memberi deskripsi mengenai gambar. Selain untuk membantu SEO website, atribut ini juga membantu memberi informasi jika gambar gagal dimuat.

<!doctype html>
<html>
  <head>
    <title>Coba Gambar</title>
  </head>
  <body>
    <img src="logo2.png" alt="Gambar Logo" />
  </body>
</html>

Perhatikan pada contoh di atas, sebelumnya gambar yang kita muat namanya logo.png, di sini kita salahkan menjadi logo2.png, karena nama file gambar tidak valid, otomatis gambar tidak bisa dimuat dan hasilnya akan digantikan dengan teks dari nilai atribut alt=" ".

6.6. Menampilkan Gambar dalam Folder yang Sama

Sebenarnya pada contoh di atas kita sudah melakukan itu, sekarang kita implementasikan ke website yang sebelumnya sudah kita buat. Ingat kita sudah pernah membuat website dengan struktur folder:

belajar-html/
│
├── index.html
│
├── pages/
│   ├── profil.html
│   └── kontak.html
│
└── images/

Sekarang kita tambah 2 file gambar di dalam folder images/ menjadi seperti berikut ini:

belajar-html/
│
├── index.html
│
├── pages/
│   ├── profil.html
│   └── kontak.html
│
├── images/
│   ├── welcome.jpg
│   └── profil.png

Di sini gambarnya hanya sebagai contoh saja, jadi bebas mau pakai gambar apa, dan nama file di dalam folder images/ menyesuaikan gambar yang digunakan.

Gambar 6.4 : contoh file gambar untuk praktik


Sekarang kita ubah kode HTML di halaman Beranda atau file index.html seperti berikut ini:

<!doctype html>
<html>
  <head>
    <title>Beranda</title>
  </head>
  <body>
    <h1>Website Saya</h1>

    <a href="index.html">Beranda</a> | <a href="pages/profil.html">Profil</a> |
    <a href="pages/kontak.html">Kontak</a>

    <hr />

    <p>Selamat datang di website saya.</p>

    <img src="images/welcome.jpg" alt="Welcome Image" />
  </body>
</html>

Dan tidak lupa ubah kode HTML di halaman Profil atau file pages/profil.html seperti berikut ini:

<!doctype html>
<html>
  <head>
    <title>Profil</title>
  </head>
  <body>
    <h1>Profil Saya</h1>

    <a href="../index.html">Beranda</a> | <a href="profil.html">Profil</a> |
    <a href="kontak.html">Kontak</a>

    <hr />

    <p>Ini adalah halaman profil.</p>

    <img src="../images/profil.png" alt="Foto Profil" />
  </body>
</html>

6.7. Menampilkan Gambar dari URL

Selain menampilkan gambar yang ada di folder lokal, kita juga bisa menampilkan gambar dari url atau link secara online.

Sebagai contoh kita modifikasi file pages/kontak.html menjadi seperti berikut ini:

<!doctype html>
<html>
  <head>
    <title>Kontak</title>
  </head>
  <body>
    <h1>Kontak</h1>

    <a href="../index.html">Beranda</a> | <a href="profil.html">Profil</a> |
    <a href="kontak.html">Kontak</a>

    <hr />

    <p>Email: admin@example.com</p>

    <img
      src="https://static.vecteezy.com/system/resources/thumbnails/021/193/715/small/circular-business-contact-communication-icon-set-png.png"
      alt="Gambar Kontak"
    />
  </body>
</html>

Gambar 6.6 : hasil gambar dari url


Dari contoh di atas, browser akan mengambil gambar langsung dari server tujuan. Kelebihan model ini adalah tidak perlu menyimpan gambar di komputer dan cocok jika gambar berada berada pada server lain. Namun kekurangannya bergantung pada koneksi internet dan jika url berubah gambar tidak dapat ditampilkan.


Untuk cara menampilkan gambar kita tidak sampai ke cara mengatur style gambar seperti ukuran, karena itu akan kita pelajari di materi tentang CSS. Jika ingin mendapatkan kode di atas bisa unduh di link berikut : 6-1-gambar.
